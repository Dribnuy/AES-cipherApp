# Encryption App

Цей проєкт — веб-додаток для шифрування та дешифрування відеофайлів прямо у браузері, використовуючи Web Workers для високої продуктивності.

## Технології
- **Frontend:** React + Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Security:** AES (Custom implementation)

## Як запустити
1. Встановіть залежності:
   
   npm install
2. Запустіть проєкт npm run dev
